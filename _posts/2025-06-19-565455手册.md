## 一些技术细节

文档使用了Sphinx来构建，通过一些Markdown或者reStructuredText来编写内容，渲染成html文档，本来想托管在`ReadTheDocs`网站上，但最后选择独立构建并传到个人服务器。

侧边栏的平滑动画和引用框样式就是自己做的代码如下

```css
/* 移动端 侧边栏动画 */
.wy-grid-for-nav{
    overflow-x: hidden;
}
.wy-nav-side{
    transition: left 0.5s;
}
@media screen and (max-width: 768px) {
    .wy-nav-content-wrap {
        margin-left: 0;
        left: 0;
        position: relative;
    }
}
.wy-nav-content-wrap{
    transition: left 0.5s;
}
/* 引用内容样式 */
.rst-content blockquote {
    position: relative;
    color: gray;
}
.rst-content blockquote:before{
    content: "";
    position: absolute;
    width: 10px;
    height: 100%;
    background-color: gainsboro;
    left: -24px;
}
```

本地构建Sphinx，Jekyll，提供给读者参考。
